Automatic Number Plate Recognition (ANPR) is the ability to automatically recognize the symbols contained in the number plates of a motor vehicle when read from an image provided by video surveillance cameras for the purposes of further processing by a security system.
First the number plate is augmented from the image using connected component analysis. The individual characters from the number plate are also segmented using connected component analysis. A convolutional neural network is trained and is used to recognize the characters.
